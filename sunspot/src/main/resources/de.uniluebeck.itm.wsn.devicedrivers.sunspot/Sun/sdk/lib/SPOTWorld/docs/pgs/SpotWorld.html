Using Solarium

Solarium is a Java&#153application that can be used to remotely manage a network of SPOTs. With Solarium, you can discover nearby SPOTs and manage the complete life-cycle of applications running on those devices. Here is some of what Solarium has to offer. Please familiarize yourself with some background terminology before initiating this exploration.

Discovering and displaying SPOTs
 

Solarium can discover and display SPOTs that are connected to the desktop via USB or can be reached via radio communication. Learn more.


Interacting with SPOTs
 

Solarium can be used to load and unload software on a SPOT, start/pause/resume/stop applications and query the current state of a device, e.g. memory usage and energy statistics. Learn more.


Managing a Network of SPOTs
 

Solarium provides a special Deployment View to make it easier to manage a network of Sun SPOTs. A deployment specifies what application should be loaded on each SPOT and allows you to deploy those applications with a single button press. The Deployment View also shows the current status of each SPOT. A deployment can also specify any associated host applications that need to be run. Learn more


Emulating SPOTs
 

Solarium also includes an emulator that can be used to run applications on a virtual  Sun SPOT. Learn more.


Robot Emulator
 

As part of the Sun SPOT team collaboration with educators we have added to Solarium an emulator for the iRobot Create. Learn more.
