What's new in Solarium?

Here are a few of the major improvements in the latest release of Solarium.

New Robot Emulator
 

As part of the Sun SPOT team collaboration with educators we have added to Solarium an emulator for the iRobot Create. Learn more


Visualize the Radio Connectivity of a Network of SPOTs
 

When debugging an application running on a network of SPOTs it can be quite helpful to be able to visualize the radio connectivity, i.e. which SPOTs can talk to each other. The Radio View provides a way for Solarium to discover and graphically depict the radio topology. Learn more


Managing a Network of SPOTs
 

Solarium provides a special Deployment View to make it easier to manage a network of Sun SPOTs. A deployment specifies what application should be loaded on each SPOT and allows you to deploy those applications with a single button press. The Deployment View also shows the current status of each SPOT. A deployment can also specify any associated host applications that need to be run. Learn more
